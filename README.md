Para empezar a realizar nuestro primer "Hola Mundo" con Spring Boot, iniciaremos diriguiendonos a Spring Boot y nos diriguimos a Learn y dentro de ella podremos encontrar diferentes opciones pero la que nos
interesa es Quickstart donde ahi podremos encontrar una guia en donde podremos encontrar como realizar nuestro Primer Hola Mundo con Spring Boot la cual luego de haber podido encontrar este tutorial nos
llegamos a diriguir a Spring Initializar en donde podremos configurar todo nuestro demo que utilizaremos para poder empezar a programar, la cual en la guia encontramos una imagen de configuración que estaremos 
utilizando sin antes tambien llegaremos a añadir dependencias que en este caso utilizaremos Spring Web la cual nos permitira poder ver lo que hemos avanzado en la nube de manera local la cual sera nuestro Primer
Hola Mundo que estamos realizando, siendo así que lo acabamos de configurar y le daremos a Generar la cual nos llegara a descargar un archivo que esta comprimido lo cual es el demo siendo asi que lo descargamos
lo buscaremos en nuestros archivos y lo descomprimimos con click derecho y "extraer en......" y elegimos donde lo extraemos para guardarlo, una vez llegado a extraerlo nos llegamos a diriguir a uno de los programas
que nos ayudara a editar código ya sea Visual Studio Code o Intellij Idea, una vez dentro de una de ellas nos diriguimos a "src/main/java/vallegrande.edu.pe.demo/PieroSoteloApplication"(donde es donde ahorita se
encuentra el código de nuestro primer "Hola Mundo", siendo así dentro seguiremos la guia de QuickStart ahi podremos seguir viendo el código de como llegar a hacer nuestro Primer Hola Mundo, una vez acabado de escribir
el código llegamos al final para ejecutar, en la terminal cuando lo ejecutemos podremos observar esto:
